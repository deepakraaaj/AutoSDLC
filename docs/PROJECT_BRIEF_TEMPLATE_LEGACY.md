# Project Brief Template

Use this template for brand-new projects or major new features when you want a standardized Markdown brief for AutoSDLC.

How to use it:
- Replace the bracketed placeholders with your project details.
- Keep the section headings the same across projects for consistency.
- If something is unknown, move it to `Open Questions` instead of inventing details.
- Save the file as `PRD.md`, `project-brief.md`, or any other `.md` name, then upload it in the app.
